6.關於這份申請網頁，請分享一個你開發時的技術心得。

趁這次的報名 我看了一下網路上金魚系列的Bootstrap快速入門，發現會使用Bootstrap框架可以很快速做出一個簡單乾淨整齊的一個網頁，也難怪做前端的都會想要學Bootstrap。

另外在Git部分當初參考彭彭老老師的"Git與GitHub專案管理工具"> ，透過command line可以快速部署到Github上，不過在"分支"部分在一開始不熟的情況下command line會比較不容易理解，如果使用sourcetree這種的GUI工具 有圖可以看相對會比較容易理解。

另外之前公司官網的source code這次將原本放在AWS EC2轉到GCP Compute Engine並將原本使用nginx利用docker部署到docker裏面並利用-v 參數將GCP裡的VM source code 部署到docker裡面的/var/www/html路徑下 作爲可快速部署與備援到其他平台。
